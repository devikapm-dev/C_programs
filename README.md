# C Programs 💻

A collection of basic C programs written during my BCA course.

## Programs
-  calculator.c — Simple calculator with switch case
-  fibonacci.c — Fibonacci series generation
-  factorial.c — Factorial using recursion
-  linear_search.c — Linear search algorithm
-  bubble_sort.c — Bubble sort algorithm
-  pattern.c — Star pattern printing

## How to compile and run
gcc filename.c -o filename
./filename

## Concepts covered
- Functions and recursion
- Arrays
- Loops
- Switch case
- Sorting and searching algorithms
